Door, also known as Heaven's Door, is an English skill Stand which does not have a Submissiveness requirement to awaken. It is currently the only Stand which can be awakened based on a School skill. It is an ability-only type Stand, meaning it can only apply its ability in combat as opposed to taking physical actions.

Overview[edit]

Door has the characteristics of an ability-type Stand, meaning it lacks any real physical capability in exchange for possessing a powerful skill.

  • Destructive Power: 0 (None)
  • Speed: 6 (B)
  • Range: 6 (B)
  • Precision: 4 (C)
  • Potential: 10 (S)

Ability[edit]

Door has the ability to transform all of your opponents in a combat scenario, consensual or non-consensual, into books; this is called 'Book Transmutation'. Doing this will allow you to 'read' them, giving you information on their physical abilities, physical characteristics, insecurities and intentions. It will also render the targets unconscious for one turn, causing them to take no actions. However, Door also allows you to 'write' one command onto each target as well. These commands can be:

  • Can't hurt me: This command will make it impossible for the target to take any damaging actions against the player. They can still strip, bind, molest and rape the player, but they cannot, for example, punch or kick them.

  • Can't cum before me: This command will make it impossible for the target to cum before the player character does.

  • Can't resist me: This command will make it impossible for the target to refuse any requests the player character makes during an encounter, such as not having anything inserted into them.

  • Will cum immediately: This command will set the target's arousal to the maximum threshold, causing them to automatically orgasm one turn after the command is applied.

  • Can't use your genitals: This command makes it impossible for the target to use their genitals in the encounter.

  • Can't write on me: This command makes it impossible for the target to try and write anything on the player character.

  • Can't touch my genitals: This command makes it impossible for the target to carry out any actions involving the player character's genitals during the encounter.

This ability to give commands is called 'Safety Lock'.

Door's ability has two primary weaknesses: one, on the turn Book Transmutation is activated, the player can take no other actions. If they attempt to take any other actions besides activating the ability, those actions will occur and the ability will fail to trigger. The second weakness is that the ability can only be used once per encounter, meaning only one command can be applied to a target. However, it is possible to give every target in an encounter a different command, or to give them all the same command, or any combination thereof, so this is much less of a weakness.

Acquisition[edit]

Door can be acquired by selecting 'School Skills' during Stand awakening, with an English skill of B or greater. It may become more difficult to acquire in future updates, but as of DoBL version 0.1.5 GIT, it is the only Stand that can be acquired using School Skills, and therefore has a 100% chance of awakening if School Skills are the only selected skill.

Mastery[edit]

As of DoBL version 0.1.5 GIT, Door does not currently possess a Mastery system.

Trivia[edit]

  • Due to its ability to make hostile opponents completely impotent via 'Can't hurt me' or to instantly put any target on a one-turn orgasm counter with 'Will cum immediately', Door allows the player character to win effectively any encounter in the game as either a Defiant or a Submissive character with equal ease. This, combined with its ease of acquisition, makes it one of the best Stands currently implemented.

  • During the History Class pillory event, Door cannot be used on the attacking student, even if no other actions are selected. It is unknown if this is a bug or a feature, as it may be the case that binding the player character's hands is supposed to disable the Book Transmutation ability.
